(Carrier robot ROS packages for a drone-mobile robot platform)

TL;DR: Carrier is a ROS-based platform where an autonomous ground robot transports, charges, and deploys a drone for last-mile scouting over rugged terrain.

Drones and mobile robots are useful for unmanned patrol and military scouting, but each platform has a different weakness: ground robots struggle in rugged terrain, while drones are limited by onboard battery capacity. Carrier combines both systems so the mobile robot can move with a larger battery, carry and charge the drone, then release it for the final scout segment.

System Focus

Integrated autonomy

Localization, perception, detection, planning, charging, and docking tasks are connected through ROS.

Hybrid mobility

The ground carrier handles long-range transport while the drone handles terrain that blocks wheels.

Simulation support

Gazebo worlds, robot descriptions, RViz configs, and launch files support development before deployment.

Mission interfaces

Custom messages, services, web interface nodes, battery fusion, teleoperation, and map export utilities are included.

Hardware System

The platform is built around a commodity mobile robot base and drone, with a mechanical interface for carrying, charging, docking, and separation during scout missions.

Architecture

ROS Stack

Carrier ROS packages
Description URDF, xacro, meshes, and RViz model configuration.
Gazebo Simulation launch files, worlds, robot model integration, and environment assets.
SLAM and Map Export GMapping launch/config files, station maps, occupancy map saving, and map cropping tools.
Navigation Move base, AMCL, EKF localization, indoor/outdoor parameters, and RViz navigation views.
Perception and Sensors LiDAR, IMU, RealSense camera launch files, floor detection, sensor fusion, and environment perception.
Interfaces Battery aggregation, ArUco support, web interface nodes, custom messages, and services.
